What is Average?

The average is a value that represents the central or typical value of a set of numbers. It is calculated by adding all numbers and dividing the total by the number of values.

How to Calculate Average?

The formula for average is:

Average = Sum of all values ÷ Number of values

For example, the average of 10, 20, and 30 is: (10 + 20 + 30) ÷ 3 = 20
